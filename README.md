# IPL Winner Prediction Using Machine Learning

This project predicts the winner of IPL matches using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
- IPL Matches dataset
- IPL Deliveries dataset

## Model
- Logistic Regression

## Output
The system predicts the winning team based on match conditions.
